# TrainingPrograms
Programs i wrote while learning Python. Programas que escrevi para aprender Python. Temática de Personal Trainer. Alguns estão incompletos. Fiz eles em 2022.

* PreTreino: Programa que eu mais utilizei na vida real. Recebe dados do cliente e retorna variáveis de treino. Pergunta o método de cálculo de vo2máx, retorna % de gordura, Massa Magra, limiares de Frequência Cardíaca de Treino de acordo com o objetivo.

* Aerobio : descobrir quantos minutos de exercício aeróbio uma pessoa precisa realizar após o treino de força para déficit calórico. Dados que ele pede: massa, ingesta calórica, FC de repouso, FCmáx, Vo2 Máx, TMB. Dados do treino: FC e minutos. Retorna o tempo de treino para o gasto calórico restante necessário.

* Borgvo2: Programa que pega os dados: massa, FC (Frequência Cardíaca) de repouso, FC máxima, Borg (nível de esforço), vo2máx, tempo de treino, TMB (Taxa metabólica Basal) e a ingesta calórica diária. Retorna gasto calórico por minuto, limiares aeróbio e anaeróbio da FC, gasto calórico do dia, déficit calórico. Classifica o tipo de metabolismo.

* FCReserva: Programa que calcula as Frequências Cardíacas para limiares aeróbio e anaeróbio. Recebe idade, massa, Frequência Cardíaca de Repouso, Frequência cardíaca máxima.

* Gastotreino: Calcula as calorias gastas em um treino funcional. Recebe nível de esforço percebido (Borg), massa, ingesta calórica, vo2máx, FC de repouso, FC máxima, Taxa Metabólica Basal e tempo de treino.

* GorduraMM: programa para calcular % de gordura e de massa magra. Incompleto.

* LimiaresMetabolismo: Recebe o nome do exercício, e o tempo que levou para a pessoa cansar de fazer ele. Retorna qual a principal rota metabólica utilizada e principal função. Pode ser rodado até usuário digitar n para sair. Para melhorar poderia escrever os dados em algum lugar para comparação futura.
